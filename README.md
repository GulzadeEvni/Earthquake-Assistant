## Get started
1. Clone Repo
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies

   ```bash
   npm install
   ```
3. Add HuggingFace Token
   ```
   // utils/ai.ts
   const HF_ACCESS_TOKEN = "YOUR_HUGGINGFACE_TOKEN_HERE";
   ```
4. Start the app

   ```bash
   npx expo start
   ```
5. Scan QR Code
